# ChestXrays

This API detects and classifies diseases from chest xray images and outputs a prediction with a label and a boundingbox

---

## Overview

This project provides a frontend for easy use, a backend that contains the model and makes predictions and a REST API for the communication between the frontend and the backend.

This allows users to:

- Send input data in image format
- Send back predictions in image format

---

## How to use

Clone the repository:

```bash
git clone https://github.com/Chest-Xray/AML.git
cd <your-project-folder>
````

Install dependencies:

```bash
pipenv install -r requirements.txt
```

Starting the server:
```bash
dockerstuff which is not ready yet
```

access the server:
go to https://localhost:8080
